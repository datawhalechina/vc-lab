# 第二章 设计原则

## 2.1 对比原则

对比是设计中最强大的原则之一，它通过差异来吸引注意力。

### 2.1.1 对比的类型

- **大小对比**：元素尺寸的差异
- **颜色对比**：明暗、冷暖的差异
- **形状对比**：几何形状的差异

### 2.1.2 对比的应用

对比可以用于：
- 突出重要信息
- 创造视觉层次
- 引导用户视线

## 2.2 对齐原则

对齐确保设计元素在视觉上保持一致的排列。

### 2.2.1 对齐方式

- **左对齐**：最常用的对齐方式
- **右对齐**：创造独特的视觉效果
- **居中对齐**：适用于标题和重要信息

### 2.2.2 对齐的重要性

良好的对齐能够：
- 提高可读性
- 增强专业感
- 建立视觉秩序

## 2.3 重复原则

重复通过重复使用相同或相似的元素来创造一致性。

### 2.3.1 重复的应用

- **颜色重复**：统一的配色方案
- **形状重复**：相似的图形元素
- **排版重复**：一致的字体和大小

## 2.4 亲密性原则

亲密性原则将相关的元素组合在一起，不相关的元素分开。

### 2.4.1 亲密性的好处

- 组织信息结构
- 减少视觉混乱
- 提高信息传达效率

## 2.5 小结

本章介绍了设计的四大基本原则：对比、对齐、重复和亲密性。掌握这些原则是创建有效设计的关键。
