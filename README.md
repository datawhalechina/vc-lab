# 设计课程-第一期（⚠️ Alpha内测版）

> ⚠️ Alpha内测版本警告：此为早期内部构建版本，尚不完整且可能存在错误，欢迎大家提Issue反馈问题或建议。

本课程旨在帮助学习者系统掌握设计基础理论与实践技能。

## 项目受众

本课程面向对设计感兴趣的初学者，包括但不限于：
- 想要入门设计领域的学生
- 希望提升设计能力的产品经理
- 想要转行做设计的从业者
